Here is a well-structured and efficient implementation of the **Merge Sort** algorithm in Java. The implementation adheres to best practices, such as avoiding integer overflow in the mid calculation, handling edge cases, and maintaining in-place sorting with auxiliary arrays for merging.

### ✅ **Java Implementation of Merge Sort**

```java
public class MergeSort {

    /**
     * Public method to initiate merge sort on an array.
     * @param array The input array to be sorted.
     */
    public void mergeSort(int[] array) {
        if (array == null) {
            return; // Handle null input gracefully
        }
        mergeSort(array, 0, array.length - 1);
    }

    /**
     * Recursive method that divides the array into halves and merges them.
     * @param array The array to be sorted.
     * @param left  The starting index of the subarray.
     * @param right The ending index of the subarray.
     */
    private void mergeSort(int[] array, int left, int right) {
        if (left >= right) {
            return; // Base case: subarray of length 1 is already sorted
        }

        // Avoid integer overflow
        int mid = left + (right - left) / 2;

        // Recursively sort the two halves
        mergeSort(array, left, mid);
        mergeSort(array, mid + 1, right);

        // Merge the sorted halves
        merge(array, left, mid, right);
    }

    /**
     * Merges two sorted subarrays into one sorted array.
     * @param array The array containing the two sorted subarrays.
     * @param left  The starting index of the left subarray.
     * @param mid   The ending index of the left subarray.
     * @param right The ending index of the right subarray.
     */
    private void merge(int[] array, int left, int mid, int right) {
        // Determine the sizes of the two subarrays
        int leftSize = mid - left + 1;
        int rightSize = right - mid;

        // Create temporary arrays to hold the two subarrays
        int[] leftTemp = new int[leftSize];
        int[] rightTemp = new int[rightSize];

        // Copy data to the temporary arrays
        for (int i = 0; i < leftSize; i++) {
            leftTemp[i] = array[left + i];
        }
        for (int j = 0; j < rightSize; j++) {
            rightTemp[j] = array[mid + 1 + j];
        }

        // Merge the temporary arrays back into the original array
        int i = 0, j = 0, k = left;
        while (i < leftSize && j < rightSize) {
            if (leftTemp[i] <= rightTemp[j]) {
                array[k++] = leftTemp[i++];
            } else {
                array[k++] = rightTemp[j++];
            }
        }

        // Copy any remaining elements from leftTemp
        while (i < leftSize) {
            array[k++] = leftTemp[i++];
        }

        // Copy any remaining elements from rightTemp
        while (j < rightSize) {
            array[k++] = rightTemp[j++];
        }
    }

    // Example usage
    public static void main(String[] args) {
        MergeSort ms = new MergeSort();
        int[] arr = {5, 3, 8, 4, 2, 7, 1, 10};
        ms.mergeSort(arr);

        // Print the sorted array
        for (int num : arr) {
            System.out.print(num + " ");
        }
        // Output: 1 2 3 4 5 7 8 10
    }
}
```
