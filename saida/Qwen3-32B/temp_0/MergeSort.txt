### ✅ **Java Implementation of MergeSort**

```java
public class MergeSort {

    /**
     * Public method to initiate the merge sort on an array.
     * @param array The array to be sorted.
     */
    public static void mergeSort(int[] array) {
        if (array == null || array.length == 0) {
            return;
        }
        mergeSort(array, 0, array.length - 1);
    }

    /**
     * Recursive helper method to divide the array into halves and sort them.
     * @param array The array to be sorted.
     * @param start The starting index of the subarray.
     * @param end The ending index of the subarray.
     */
    private static void mergeSort(int[] array, int start, int end) {
        if (start >= end) {
            return;
        }

        // Avoid integer overflow
        int mid = start + (end - start) / 2;

        // Recursively sort the left and right halves
        mergeSort(array, start, mid);
        mergeSort(array, mid + 1, end);

        // Merge the sorted halves
        merge(array, start, mid, end);
    }

    /**
     * Merges two sorted subarrays into a single sorted subarray.
     * @param array The array containing the two sorted subarrays.
     * @param start The starting index of the left subarray.
     * @param mid The ending index of the left subarray.
     * @param end The ending index of the right subarray.
     */
    private static void merge(int[] array, int start, int mid, int end) {
        // Create a temporary array to hold the merged result
        int[] temp = new int[end - start + 1];

        int i = start;      // Index for the left subarray
        int j = mid + 1;    // Index for the right subarray
        int k = 0;          // Index for the temp array

        // Merge the two subarrays into temp
        while (i <= mid && j <= end) {
            if (array[i] <= array[j]) {
                temp[k++] = array[i++];
            } else {
                temp[k++] = array[j++];
            }
        }

        // Copy any remaining elements from the left subarray
        while (i <= mid) {
            temp[k++] = array[i++];
        }

        // Copy any remaining elements from the right subarray
        while (j <= end) {
            temp[k++] = array[j++];
        }

        // Copy the merged elements back into the original array
        for (int l = 0; l < temp.length; l++) {
            array[start + l] = temp[l];
        }
    }

    // Example usage
    public static void main(String[] args) {
        int[] array = {5, 3, 8, 4, 2, 7, 1, 6};
        mergeSort(array);

        // Print the sorted array
        for (int num : array) {
            System.out.print(num + " ");
        }
    }
}
```

- **Divide and Conquer Strategy**: The array is recursively split into halves until each subarray contains a single element.
- **Efficient Merging**: The `merge` function uses a temporary array to merge two sorted subarrays in linear time.
- **Avoids Integer Overflow**: The midpoint is calculated using `start + (end - start) / 2` to prevent overflow in large arrays.
- **Handles Edge Cases**: The method gracefully handles `null` or empty arrays.
- **In-Place Sorting**: The original array is modified in place, and no new array is returned.
