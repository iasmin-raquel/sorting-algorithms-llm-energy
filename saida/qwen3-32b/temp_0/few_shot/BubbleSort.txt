Sure! Here's a `BubbleSort` implementation in Java, following the same style and structure as the provided `InsertionSort` example:

```java
public class BubbleSort {
    public static void sort(int[] arr) {
        int n = arr.length;
        for (int i = 0; i < n - 1; i++) {
            // Flag to detect if the array is already sorted
            boolean swapped = false;
            for (int j = 0; j < n - i - 1; j++) {
                if (arr[j] > arr[j + 1]) {
                    // Swap arr[j] and arr[j+1]
                    int temp = arr[j];
                    arr[j] = arr[j + 1];
                    arr[j + 1] = temp;
                    swapped = true;
                }
            }
            // If no two elements were swapped in the inner loop, then break
            if (!swapped) {
                break;
            }
        }
    }

    public static void main(String[] args) {
        int[] arr = {64, 34, 25, 12, 22, 11, 90};
        sort(arr);
        for (int num : arr) {
            System.out.print(num + " ");
        }
    }
}
```

### Key Features:
- **Optimized Bubble Sort**: Includes a `swapped` flag to stop early if the array becomes sorted before all passes.
- **Same structure**: Uses a `sort` method and a `main` method with a test array and output loop, just like the `InsertionSort` example.
